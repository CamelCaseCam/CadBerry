Special keys
------------
CDB_MOD_SHIFT: 0x0001
^^^^^^^^^^^^^^^^^^^^^
Is shift down?

CDB_MOD_CONTROL: 0x0002
^^^^^^^^^^^^^^^^^^^^^^^
Is control down?
  
CDB_MOD_ALT: 0x0004
^^^^^^^^^^^^^^^^^^^
Is alt down?
   
CDB_MOD_SUPER: 0x0008
^^^^^^^^^^^^^^^^^^^^^
Are super keys pressed? (I'm not sure what this does, but it was in the tutorial)
	
CDB_MOD_CAPS_LOCK: 0x0010
^^^^^^^^^^^^^^^^^^^^^^^^^
Is caps lock on?

CDB_MOD_NUM_LOCK: 0x0020
^^^^^^^^^^^^^^^^^^^^^^^^
Is num lock on?
